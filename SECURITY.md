# セキュリティポリシー

## サポートされているバージョン

| バージョン | サポート状況 |
|-----------|------------|
| 最新版     | ✅ サポート中 |
| それ以前   | ❌ サポート終了 |

## 脆弱性の報告

セキュリティ上の脆弱性を発見された場合は、以下の手順で報告をお願いします。

1. **公開しないでください** - Issue やプルリクエストで脆弱性を公開しないでください
2. **メールで報告** - [security@contoso.com](mailto:security@contoso.com) 宛にメールで詳細をお送りください
3. **報告に含める情報**:
   - 脆弱性の種類と影響範囲
   - 再現手順（ステップバイステップ）
   - 影響を受けるバージョン
   - 可能であれば修正案

## 対応方針

- 報告受領後、**3 営業日以内**に確認の連絡をいたします
- 脆弱性の評価と修正に最大 **90 日**かかる場合があります
- 修正完了後、セキュリティアドバイザリを公開します

## 対象範囲

以下が本ポリシーの対象範囲です：

- アプリケーションコード
- AI モデル設定・プロンプト
- API エンドポイント
- 依存パッケージ

## 謝辞

セキュリティ脆弱性の責任ある報告にご協力いただいた方には、README の謝辞セクションに掲載させていただきます。
